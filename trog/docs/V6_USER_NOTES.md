# 🎲 V6 업데이트 — 2026-04-24

## 주요 변경

### 🪙 새 시스템

- **💰 소지 금액 추가.** 직업별 시작 골드(전사 50 / 마법사 80 / 도적 70 / 성직자 60). DM이 거래·전리품·여관비 등에서 자동으로 변동시킨다. 캐릭터 패널 + 파티 카드에 실시간 표시.
- **🛡 장비 vs 🍶 소모품 vs 📜 퀘스트 구별.** 인벤 카드에 종류 뱃지 + 버튼이 종류별로 다름.
  - 소모품 → "사용" 버튼 (2번 눌러야 확정)
  - 장비 → "장착" 버튼 (기존 장비는 자동으로 인벤으로 회수)
  - 퀘스트 → 사용·장착 불가 (서사용)
  - DM이 종류 명시 안 하고 줄 수도 있음 — 그 땐 그냥 "사용" 누르면 서버가 "장비입니다, 장착할까요?" 물어봄.
- **🌫 관망 / 진행 버튼.** 행동 없이 DM이 장면을 한 단계 더 진척시키고 싶을 때. 파티가 잠시 상황을 지켜보는 자동 액션이 들어가고 DM이 다음 전개를 풀어준다.
- **⏭ 내 턴 패스 버튼.** 위와 다른 점: **LLM 호출 없이** 그냥 본인 턴만 다음 사람에게 넘김. (방장의 강제 스킵과 달리 본인만 본인 턴을 넘길 수 있음 — 다른 사람 마음대로 못 넘김)

### 🔧 기존 문제 수정

- **📱 모바일에서 아이템 효과 보기.** 데스크톱은 마우스 hover 로 보였지만 모바일은 안 됐음. 이제 **인벤 카드를 탭하면** 효과가 펼쳐짐.
- **🧼 디버프 즉시 해제 인식.** "정화 주문으로 독을 풀어줬다" DM 서사가 떠도 디버프가 정직하게 턴만 줄어들던 문제. 이제 DM이 `[이름 상태 해제: 효과명]` 태그를 찍으면 즉시 사라진다.
- **🐺 동물 울음소리.** "끄아악-!" 으로 죽던 개구리가 이제 "꽉-!" 또는 "끅-…" 식으로 그 종 특유의 발성으로 죽도록 DM 가이드 강화.
  ※ 이건 LLM 톤 가이드라 가끔 무시될 수 있음.
- **⚖ 인원 수에 따른 난이도 조절.** DM 프롬프트 첫 줄에 `[파티 N명 · 평균 LvX.X]` 가 박힘. DM이 이를 보고 적의 수·HP·강도를 조절. 1~2 명은 평이, 5+ 명은 우두머리 단일전 또는 다수+정예 혼성.
- **🏘 마을 도착 시 선택지 유도.** 파티가 평화 지역에 도착하면 시야에 들어오는 시설 2~4개 가벼운 묘사 + 거리의 소문/게시판 의뢰 한 토막. "무엇을 하시겠습니까?" 같은 직접 질문은 금지 — 자연스러운 묘사로 고리만.
- **💀 자해 행동 시 본인 디버프.** 본인이 독 묻은 단도를 핥으면 본인에게도 독 디버프가 걸리도록 DM 가이드. (역시 LLM 가이드라 가끔 무시 가능)
