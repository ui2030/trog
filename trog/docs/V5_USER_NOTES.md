# 🎲 V5 업데이트 — 2026-04-24

> "원래 있어야 했던 것들" 모음. 눈에 크게 띄지 않아도 플레이할 때 "어? 이거 이상한데" 싶던 것들을 한 번에 잡았다.

## 주요 변경

### 🎮 게임성

- **레벨업해도 이젠 '풀피 부활' 아님.** HP 5 에서 레벨업해도 15 가 됨. 증가분만 더해지므로 DM 이 "간신히 살아남았다" 서술해도 수치가 어긋나지 않는다.
- **드디어 밤이 지나간다.** 전엔 심야 한 번 찍히면 영영 심야. 이제 자연스럽게 새벽으로 넘어가면 서버가 "**2일차**" 로 인식하고 서사에 반영.
- **3턴 버프 = 본인 차례 3번.** 전엔 파티원 4명이면 1라운드도 못 버티고 사라졌다. 이제 내가 행동할 때만 1턴 줄어든다.
- **연결 끊기면 "턴 스킵 → 90초 뒤 이탈" 두 번 안 뜸.** grace 90초가 지난 시점에 이탈 서사와 턴 스킵이 한 번에 묶여서 공지.
- **커스텀 그림 업로드해도 서버 무거워지지 않음.** 그림이 이제 URL 로 서빙되므로 DM 응답마다 1 MB 짜리 data URL 이 다시 내려가는 일 없음. 모바일 훨씬 쾌적.

### 🦊 수인 종족

- **퍼리 캐릭터로 나오던 문제 개선.** 90% 영역이어도 "휴머노이드 영웅" 토큰을 함께 박아 발더스게이트 톤 유지.
- **슬라이더 범위가 10%~90% 로 정리.** 0% 는 인간과 구별이 없고, 100% 는 짐승이라 "수인" 정체성이 깨지는 경계. 그 사이 값으로만 선택.
- **그림이 슬라이더를 따라 5단계로 변함.** 전엔 3단계(33/66 경계)라 슬라이더를 잔뜩 돌려도 그림이 똑같았다. 이제 25/45/55/70 로 나뉘어 조정 체감.
- **지원 안 하는 동물 입력하면 에러 명시.** 전엔 조용히 늑대로 바꿨음.

### 🔒 공정성

- **주사위는 이제 서버가 굴림.** DevTools 조작해서 항상 20 찍는 짓 불가.
- **DM 의 XP 보상 상한 도입.** 한 태그당 200, 한 응답에서 총 500. "DM 갑자기 XP 9999" 이상 현상 차단.
- **장비/아이템 효과 태그 한 번에 모두에게 걸리던 버그 수정.** `[장비 효과: 플레이어 | 장비 | 효과]` 식으로 플레이어를 지목해야 적용. 같은 장비 든 사람이 파티 내 1명일 때만 생략형도 허용.
- **플레이어 행동 텍스트에 `[ ]` 섞어 DM 흉내내던 주입 시도 차단.** 액션 입력의 대괄호는 전각으로 치환.

### 🏛 운영

- **휴면 상태는 24시간 후 자동 소멸.** 방 메모리 누수 방지.
- **"휴면 잠금 해제" 는 2단 확인.** 방장이 실수로 잠깐 끊긴 친구 캐릭터를 다른 사람에게 넘기는 것 방지.
- **끊긴 사람이 새 방장이 되는 버그 수정.** 연결된 사람 중에서만 승계. 후보가 없으면 "방장 없음" 상태가 됐다가 다음 입장자에게 자동 위임.
- **DM 응답이 30초 안에 안 오면 타임아웃.** 전엔 API 가 멍때리면 방 전체가 멈췄다.
