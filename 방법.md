## 📁 폴더가 뭐가 뭔지

```
trpg/
├── trog/                           ← 진짜 게임 서버. 대부분의 작업은 여기서.
├── claude-code-openai-wrapper/     ← 선택. 구독으로 Claude 쓰고 싶을 때만 필요.
├── 방법.md                         ← 지금 이 파일
└── FEATURE_UPDATE_*.md             ← 과거 개발 노트 (무시해도 됨)
```

### trog
- FastAPI + WebSocket 게임 서버. **포트 8080**.
- `.env`에 Anthropic API 키 넣고 `python main.py` 하면 끝.

### claude-code-openai-wrapper
- **Claude Pro/Max 구독**을 OpenAI 호환 API처럼 노출시키는 프록시. **포트 8000**.
- API 키 비용 아끼고 싶을 때만 씀. 안 쓰면 폴더 통째로 무시해도 됨.
- Windows는 반드시 `run_win.py`로 실행 (asyncio 충돌 회피).

## ▶️ 매일 띄우는 법

### 시나리오 A — 친구랑 플레이 (기본)

**터미널 1개**만 있으면 됨.

```bash

cd c:\Users\ui2030\Documents\trpg\trog

```

콘솔에 뜨면 성공:
```
INFO:     Uvicorn running on http://0.0.0.0:8080
```

→ 브라우저 열고 **http://localhost:8080** 접속.

### 시나리오 B — 친구 외부 네트워크에서 (Cloudflare 터널)

터미널 **2개**.

**터미널 1 — 게임 서버** (위 시나리오 A 명령 그대로):
```bash
conda activate Trpg
cd c:\Users\ui2030\Documents\trpg\trog
python main.py
```

**터미널 2 — 터널**:
```bash
cd c:\Users\ui2030\Documents\trpg\trog
tunnel.bat
```
콘솔에 뜨는 `https://xxxx-xxxx-xxxx.trycloudflare.com` 을 친구에게 공유.

> **cloudflared 없다**는 에러 나면 최초 1회 설치:
> ```
> winget install --id Cloudflare.cloudflared
> ```
> 설치 후엔 **새 터미널** 열어야 PATH 잡힘.

### 시나리오 C — 혼자 테스트 (wrapper, 크레딧 안 씀)

터미널 **2개** + `.env`의 `LLM_MODE=wrapper` 로 변경.

## 🩺 안 될 때 (증상별)

### ❌ `ImportError: cannot import name 'Undefined' from 'pydantic.fields'`
**conda 환경을 안 켜서 base 환경으로 돌아간 거.** 해결:
```bash
conda activate Trpg
```
그 다음 `python main.py` 다시.

### ❌ `python: command not found`
conda 환경이 안 켜짐 또는 Python 자체 미설치. 먼저 `conda activate Trpg`.  
그래도 안 되면 https://www.python.org/downloads/ 에서 3.11+ 설치.

### ❌ `ModuleNotFoundError: No module named 'fastapi'`
`Trpg` 환경에 의존성 누락:
```bash
conda activate Trpg
cd c:\Users\ui2030\Documents\trpg\trog
pip install -r requirements.txt
```

### ❌ 게임 들어갔는데 "⚠ DM 호출 실패: authentication_error"
`.env`의 `ANTHROPIC_API_KEY`가 틀림. 복사하면서 공백/오타(y, s 중복) 없나 확인.

### ❌ "⚠ DM 호출 실패: credit balance too low"
Anthropic 크레딧 0. https://console.anthropic.com/ → Billing → 크레딧 충전.  
또는 wrapper 모드로 전환 (시나리오 C).

### ❌ (wrapper 모드) "⚠ DM 호출 실패: Connection error"
wrapper 서버(8000)가 안 떠있음. 터미널 1에서 `python run_win.py` 다시 실행.

### ❌ "방을 찾을 수 없습니다"
서버 재시작하면 방 전부 증발. 새로 만들면 됨.

### ❌ 화면이 예전 버전
강제 새로고침: **Ctrl + F5**

### ❌ "너무 빠릅니다. X초 뒤에 다시 시도하세요"
3초 쿨다운 (레이트리밋). 잠시 기다리기.

### ❌ 내 차례가 아니라는 메시지
TRPG라 턴제. 입장 순서대로 돌아가면서 행동. 친구 차례 기다리기.

### ❌ 진행 중인데 캐릭터 다시 만들고 싶음
`http://localhost:8080?fresh=1` 접속하거나, 오른쪽 하단 `🚪 방 나가기 / 새 캐릭터` 버튼.

## 🎮 게임 중 팁

- **플레이어 이름이 톤에 영향** — "허접"으로 만들면 NPC들이 비꼼, "강철왕"은 서사시 톤.
- **그림 그리기** — 🎨 버튼 누르면 캔버스. 그리면 다음 DM 응답에 외양이 반영됨.
- **커스텀 퀵액션** — "+ 나만의 행동"으로 자주 쓰는 문구 저장 (우클릭으로 삭제).
- **주사위** — 하단 `d4 d6 d8 d10 d12 d20 d100` 직접 굴림. d20 에서 20은 크리티컬, 1은 펌블.
- **관전자 모드** — 엔트리 화면 하단 "관전자 모드"로 방 코드만 넣고 입장. 채팅만 가능, 턴엔 영향 없음.
