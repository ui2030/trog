# 🚀 TROG 처음부터 끝까지 따라하기

친구랑 플레이 준비 완료하는 데까지 3분. 터미널 2개 여는 가이드.

---

## 준비물 체크

딱 2개만 확인하면 됨:

- ✅ `Trpg` conda 환경 있음 (있음 — 이미 설정됨)
- ✅ Anthropic API 키 `.env`에 들어있음 (있음 — 확인됨)

둘 다 OK면 아래 4단계만 따라가면 돼.

---

## 1️⃣ 터미널 **2개** 열기

**Win + R** → `cmd` 입력 → **Enter** → 창 하나 뜸.  
**한 번 더 반복해서 총 2개** 열어둠.

> 💡 하나는 **게임 서버**, 다른 하나는 **터널**용. 둘이 같이 있어야 친구가 들어올 수 있음.

---

## 2️⃣ 터미널 ①: 게임 서버 켜기

**복붙:**
```
conda activate Trpg
cd c:\Users\ui2030\Documents\trpg\trog
python main.py
```

### ✅ 성공 신호
화면에 다음 줄이 뜨면 OK:
```
[LLM] mode=anthropic  model=claude-sonnet-4-6
INFO:     Started server process [xxxxx]
INFO:     Uvicorn running on http://0.0.0.0:8080 (Press CTRL+C to quit)
```

이 창은 **닫지 말고 그대로 둠**. 닫으면 서버 꺼짐.

### ❌ 이런 에러 뜨면

| 에러 메시지 | 원인 / 해결 |
|---|---|
| `ImportError: cannot import name 'Undefined'` | `conda activate Trpg` 빼먹음. 다시 쳐. |
| `ModuleNotFoundError` | 패키지 누락. `pip install -r requirements.txt` |
| `(Trpg)` 프롬프트 안 뜸 | `conda activate Trpg` 가 먹히지 않음. Anaconda Prompt로 열기 |

---

## 3️⃣ 터미널 ②: Cloudflare 터널 켜기

**복붙:**
```
cd c:\Users\ui2030\Documents\trpg\trog
tunnel.bat
```

### ✅ 성공 신호
조금 기다리면 이런 프레임이 뜸:
```
+-----------------------------------------------------------+
|  Your quick Tunnel has been created! Visit it at:         |
|  https://XXXXX-XXXX-XXXXX.trycloudflare.com              |
+-----------------------------------------------------------+
```

**이 URL이 친구한테 줄 주소.** 복사해서 카톡으로 전달.

이 창도 **닫지 말고 그대로 둠**. 닫으면 URL 소멸.

### ❌ 이런 에러 뜨면

| 에러 메시지 | 원인 / 해결 |
|---|---|
| `cloudflared: command not found` | 최초 1회 설치 필요 → `winget install --id Cloudflare.cloudflared`, 그 후 **새 터미널** 열어서 재시도 |
| `tunnel.bat이(가) 내부 또는 외부 명령...` | `cd` 를 잘못 침. `c:\Users\ui2030\Documents\trpg\trog` 에서 실행하는 게 맞음 |

---

## 4️⃣ 내 브라우저에서 테스트

**주소창에 입력:**
```
http://localhost:8080
```

TROG 화면 뜨면 성공. 방 만들어서 코드 확인하고 시작.

친구는 **3단계에서 받은 `trycloudflare.com` URL**로 접속.  
너는 로컬(`localhost:8080`)로도, 그 URL로도 둘 다 접속 가능.

---

## 🎮 플레이 시작

1. 이름 입력 → 직업 선택 → `새 방 만들기`
2. 6자 **방 코드** 뜸 (예: `F5T2K3`)
3. 친구한테 **URL + 방 코드** 둘 다 보내기
   ```
   https://XXXXX-XXXX-XXXXX.trycloudflare.com
   방 코드: F5T2K3
   ```
4. 친구들 전부 입장하면 각자 `⚔️ 모험 시작!` 눌러서 준비
5. 전원 준비되면 자동으로 게임 시작 (DM이 오프닝 씬 서술)

## ❓ 자주 하는 실수

### 🔁 "어제 URL이 오늘 안 됨"
Quick tunnel은 매번 **완전히 새 URL** 발급. 터미널 ② 켤 때마다 새 주소 뜨는 거 확인하고 친구한테 **그때그때** 공유.  
고정 URL은 Cloudflare 계정 + 도메인 + named tunnel 필요 (별도 설정).

### 🧭 "터미널 ①만 켜면 친구가 못 들어옴"
로컬(너 컴퓨터)에서만 접속 가능한 상태. **터미널 ② (터널)도 켜야** 외부에서 접근 가능.  
본인 집 와이파이로 같이 앉아있는 거면 `192.168.x.x:8080` 으로도 되지만, 다른 집이면 터널 필수.

### 💸 "DM 호출 실패: credit balance too low"
Anthropic API 크레딧 0원. https://console.anthropic.com/ → **Billing** → 크레딧 충전 (보통 $5~10).

### 🔒 "너무 빠릅니다. X초 뒤에 다시 시도하세요"
3초 쿨다운 (연타 방지). 잠깐 기다리면 됨.

### 🎯 "내 차례가 아닙니다"
TRPG라 **입장 순서대로 턴제**. 친구 차례면 너는 못 움직임. 기다리거나, 친구가 놓치면 오른쪽 하단 `⏭ 턴 스킵` (방장 전용).
